# Network Testing Automation using Python

## Description
This project automates network connectivity testing using Python. It sends ping (ICMP) requests to multiple IP addresses and checks whether the systems are reachable (UP/DOWN).

## Features
- Ping multiple IP addresses
- Detect UP/DOWN status
- Show summary of results
- Save output in a log file

## Technologies Used
- Python
- OS module

## How to Run
1. Install Python
2. Open terminal in project folder
3. Run:
   python network_test.py

## Sample Output
8.8.8.8 is UP  
1.1.1.1 is UP  
192.168.1.1 is UP  
10.0.0.1 is DOWN  

Summary:  
UP: 3 | DOWN: 1
